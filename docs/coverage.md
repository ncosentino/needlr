# Code Coverage Report

The detailed code coverage report shows test coverage for all Needlr packages.

!!! info "Coverage Details"
    This report is generated by [ReportGenerator](https://reportgenerator.io/) from test results.
    
    The coverage report shows:
    
    - **Line Coverage**: Percentage of code lines executed during tests
    - **Branch Coverage**: Percentage of code branches (if/else) tested
    - **Method Coverage**: Percentage of methods called during tests

## View Coverage Report

<a href="../coverage/" class="md-button md-button--primary" target="_blank">
    Open Coverage Report
</a>

The coverage report opens in a new tab and provides:

- Summary of all assemblies
- Detailed line-by-line coverage
- Risk hotspots and complexity metrics
- Historical coverage trends

---

*Coverage report is updated automatically on each push to main.*
